    What is Cron Expression Visualizer and Explainer?

    Cron Expression Visualizer and Explainer is a focused browser-based utility built for fast, private, task-oriented workflows.

    Why are some complex expressions rejected?

    The parser focuses on transparent browser-only logic and avoids ambiguous vendor-specific cron extensions.

    Is this tool private?

    Yes. All calculations run in your browser. The page does not upload your inputs to a server.

    Which cron format is supported?

    The tool supports common five-field cron expressions: minute, hour, day of month, month, and day of week.

    Can I use the result commercially?

    Yes. The generated output is meant for everyday operational use, but you should verify critical legal, tax, and security decisions with a qualified professional.

    Does it work offline?

    After the first load, the core JavaScript logic is self-contained. CDN styling and ads require network access.
